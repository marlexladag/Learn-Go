A structured learning journey from Go basics to mastery.

## Roadmap

### Phase 1: Foundations (Days 1-7)
- Day 1: Setup, variables, types, constants, printing
- Day 2: If/else, for loops, switch statements
- Day 3: Functions, multiple returns, variadic functions
- Day 4: Arrays and slices
- Day 5: Maps
- Day 6: Pointers
- Day 7: Review & mini project

### Phase 2: Core Go (Days 8-21)
- Days 8-10: Structs & methods
- Days 11-14: Interfaces
- Days 15-17: Error handling
- Days 18-21: Packages & modules

### Phase 3: Concurrency (Days 22-35)
- Days 22-25: Goroutines
- Days 26-30: Channels
- Days 31-35: Concurrency patterns

### Phase 4: Practical Skills (Days 36-50)
- Days 36-40: Testing
- Days 41-45: Web development
- Days 46-50: Database & files

### Phase 5: Advanced Topics (Days 51-70)
- Days 51-55: Generics
- Days 56-60: Reflection & code generation
- Days 61-65: Performance & profiling
- Days 66-70: Advanced patterns

### Phase 6: Mastery (Days 71-90+)
- Build real projects
- Contribute to open source
- Deep dive into Go internals
